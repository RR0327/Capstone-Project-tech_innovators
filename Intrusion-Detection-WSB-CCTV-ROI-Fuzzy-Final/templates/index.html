Hybrid Intrusion Detection System

Security dashboard for image, video, and live-camera monitoring. The system first detects intrusion, then checks whether the detected bounding box enters the restricted ROI zone, and finally uses fuzzy logic for the alert decision.

Model behaviour: the supplied model has one trained class: intrusion. A scene is shown as non-intrusion when no valid intrusion detection meets the decision rules.
ROI + fuzzy workflow: intrusion detection runs first. If intrusion is found, the bounding box is checked against the restricted ROI area. If it enters the ROI, fuzzy logic evaluates classes such as abnormal person, covered person, normal person, weapon, or background.

Image Detection

Upload a CCTV image. The result page shows intrusion status, restricted ROI checking, fuzzy class, risk score, and final alert decision.

Video Detection

Upload a CCTV video. The system checks intrusion frame by frame, validates ROI entry, applies fuzzy logic, and creates an annotated output video.

Live Camera

Monitor a live camera connected to this computer. The dashboard shows intrusion status, ROI entry, fuzzy class, and risk score in real time.
